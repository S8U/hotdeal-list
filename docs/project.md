# 핫딜 애그리게이터

## 개요
여러 커뮤니티의 핫딜 게시글을 크롤링하여 보여주는 애그리게이터 사이트

## 기능
1. 핫딜 데이터 크롤링
    - [x] 쿨엔조이
    - [x] 퀘이사존 핫딜
    - [x] 퀘이사존 타세요
    - [x] 클리앙
    - [x] 루리웹
    - [x] 뽐뿌
2. 핫딜 데이터 AI 가공
    - [x] 카테고리 추출
    - [x] 상품명 추출
    - [x] 상품명 영문 번역
    - [x] 제목 영문 번역
3. 핫딜 리스트 제공
    - [ ] ES search_after 기반 무한 스크롤
4. 핫딜 검색 기능 제공
    - [ ] 필터
        - [ ] 카테고리
        - [ ] 가격 범위 (KRW)
        - [ ] 커뮤니티 타입
5. 핫딜 가격 히스토리 제공
    - [ ] AI로 가공한 상품명을 ES로 검색
    - [ ] 해당 상품들의 가격과 날짜를 그래프로 제공
6. 회원가입, 로그인
    - [ ] JWT 인증
        - [ ] 추후 앱 개발 고려
        - [ ] 여러 기기에서 로그인할 것을 고려
        - [ ] 기본적으로 로그인 유지
        - [ ] Refresh Token 30일
        - [ ] 로그아웃 시 Soft Delete
    - [ ] 이메일 로그인 (기본)
    - [ ] 소셜 로그인
        - [ ] 구글
        - [ ] 네이버
        - [ ] 카카오
7. 핫딜 웹 푸시 알림 제공 (키워드 알림)
    - [ ] 로그인 후 키워드 등록
    - [ ] 로그인 후 기기별 알림 ON/OFF
    - [ ] 등록한 키워드에 맞는 핫딜이 생성될 때 웹 푸시 알림 발송

## 데이터 수집 Flow
1. 커뮤니티 게시글 리스트 크롤링
2. 커뮤니티 게시글 Raw 데이터 저장
    - 이미 저장한 게시글: 추천수, 조회수, 댓글수 Raw 데이터 업데이트
    - 아직 저장하지 않은 게시글: 게시글 상세 크롤링 및 Raw 데이터 생성
3. 커뮤니티 게시글 Raw 데이터를 AI로 가공
4. Raw 데이터 + AI 가공 데이터로 핫딜 데이터 생성 또는 업데이트
    - 핫딜 데이터 생성 시: 썸네일 다운로드
        - 우선순위
            - 1: 썸네일 이미지
            - 2: 게시글 내용 첫번째 이미지
        - 저장 옵션
          - 로컬 파일
          - Cloudflare R2 (AWS S3 API 사용)
5. 핫딜 데이터를 Elasticsearch에 저장
6. 핫딜 데이터 생성 시: 핫딜 웹 푸시 알림 발송
